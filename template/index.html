Backend no Ar!

O Serviço faz parte do Projeto de Conclusão de Curso da Universidade Federal Fluminense do Polo de Rio das Ostras.
O projeto viza fornecer um questionario a respeito das preferencias do usuario e com base nelas, recomenda uma lista de ate 10 cidades para se viver.
O projeto foi implementado principalmente usando o framework Flask para o backend e MySQL como ferramenta para o Banco de Dados.

Endpoints disponiveis

  • Ver as questões: "/info-api/questions"
  • Ver as cidades (Dados Gerais): "/info-api/cities"
  • Ver as cidades de determinado Estado (Dados Gerais): "/info-api/state/[uf]"
  • Ver informações mais detalhadas da cidade: "/info-api/city/info/[cityId]"
  • Gera a recomendação de cidades -> necessario enviar um objeto na requisição: "/info-api/city/recomendation"
